Über dieses Projekt

Diese Website wurde im Rahmen der Portfolioprüfung für das Modul "Clientseitige Programmierung" entwickelt und demonstriert moderne Webtechnologien.

Kreativität

Innovative Ideen und moderne Webentwicklung mit HTML5, CSS3 und JavaScript

Teamwork

Effektive Zusammenarbeit im Pair-Programming-Verfahren

Performance

Schnelle, responsive Webanwendungen mit Bootstrap 5

Weitere Informationen

Ziel dieses Projekts ist die Entwicklung einer professionellen, responsiven Website mit modernen Webtechnologien. Die Website demonstriert:

  • Semantisches HTML5 und barrierefreie Strukturen
  • Responsive Design mit Bootstrap 5 Grid-System
  • JavaScript-Interaktivität und Formularvalidierung
  • Prinzipien der Usability und User Experience

Frontend
  • HTML5 (semantische Tags)
  • CSS3 (Custom Properties, Flexbox, Grid)
  • JavaScript ES6+
  • Bootstrap 5.3
Tools & Libraries
  • Bootstrap Icons
  • Google Fonts
  • Responsive Images
  • Form Validation API
